# ボーダー
`bordered` を設定してテーブルボーダーを實現し、値は `true` | `false` | `h` | `v` | `body-v` です。

| 値 | 説明 |
| --- | --- |
| `true` | すべてのテーブルボーダー |
| `false` | ボーダーなし |
| `h` | 横線のみ |
| `v` | 縦線のみ |
| `body-v` | ヘッダーとボディは横線、ボディは縦線 |
| `body-h` | ヘッダーとボディは縦線、ボディは横線 |

::: tip
スクロールバーの影響により、テーブルの右端と下端のボーダーはセルの `border-right` と `border-bottom` で実装されており、消える場合があります。必要に応じてカスタムCSSを追加できます。
:::
